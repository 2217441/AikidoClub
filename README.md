## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
# Clone the repository
git clone https://github.com/2217441/AikidoClub.git
cd AikidoClub

# Install dependencies
npm install
```

### Development

```bash
# Start dev server with hot reload
npm start
```

Opens at `http://localhost:8080`

### Production Build

```bash
# Build for production
npm run build
```

Output goes to `dist/` folder.
